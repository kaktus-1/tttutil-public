# TTT Util

A Fabric client mod for Minecraft **1.21.11** featuring stats and utility functions for TTT (Trouble in Terrorist Town).

## Download

Grab the latest version from the **[Releases](https://github.com/kaktus-1/tttutil-public/releases)** page.

Just download the JAR file matching your Minecraft version (e.g., `tttutil-1.0.8-1.21.11.jar`) and drop it into your `.minecraft/mods/` folder.

## Requirements

- Minecraft 1.21.11
- Fabric Loader 0.19.3+
- Fabric API
